Auditium

AI-powered website audit engine delivering instant compliance, performance, and security insights.

---

## 🚀 Overview

Auditium is an automated web auditing platform designed to analyze websites for:

- ⚡ Performance
- 🔐 Security
- ♿ Accessibility (WCAG)
- 📋 Compliance standards
- 📊 Technical best practices

The system generates structured reports with scoring and insights.

---

## 🛠 Tech Stack

- HTML / CSS / JavaScript (Frontend)
- n8n (Workflow Automation)
- AI-based analysis engine
- Automated email reporting

---

## 📦 Features

- Website URL analysis
- Automated audit scoring
- Structured JSON reports
- Email report delivery
- Modular audit architecture
